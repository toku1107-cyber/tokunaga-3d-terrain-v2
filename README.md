# (有)徳永測量 社屋 3D地形ビューア VER2（広域1km）

UAV写真点群 + 国土地理院DEM5a + 筆界データ（登記所備付地図）で構築した広域1km四方の3D地形ビューア。

## 公開URL
https://toku1107-cyber.github.io/tokunaga-3d-terrain-v2/

## 特徴
- **広域1km × 1km**（VER1は社屋中心92.8×90.1m）
- **ハイブリッド標高**: UAV LAZ 中心部 + 国土地理院DEM5a 外周部
- **筆界データ131筆**（小林市法務局 登記所備付地図）
- 国土地理院空中写真タイル（z=16）背景

## データ
- LAZ点群: 6,231,929点（中心7,600セル・0.7%）
- DEM5a: 1,094,900セル（99.3%・z=15タイル合成）
- 筆界: 131筆・5,129線分・131ラベル
- グリッド: 1050×1050
- 標高範囲: 186.76〜286.16m

## 座標系
EPSG:6670 JGD2011 平面直角II系
